Simple Addition Program

Description
This C++ program performs a simple addition operation and displays the result in a formatted manner.

Features
- Takes two predefined integers as input
- Displays the numbers and the addition operator in a formatted manner
- Calculates and displays the sum of the two numbers
- Waits for user input before exiting

Code Structure
The code consists of a single main function, which contains the entire program logic.

Dependencies
- iostream: For input/output operations
- conio.h: For _getch() function to wait for user input

Compilation and Execution
To compile and run the program, save it in a file with a .cpp extension (e.g., addition.cpp) and use a C++ compiler.
